# Tabla Periódica Interactiva

Tabla periódica de los elementos completa e interactiva, pensada como recurso educativo para secundaria. Incluye los 118 elementos con información detallada, descripciones didácticas y ejemplos cotidianos.

**[Ver en vivo](https://javicanton.github.io/tabla-periodica/)**

## Características

- **118 elementos** con número atómico, símbolo, nombre, masa atómica, configuración electrónica y estado de la materia.
- **Modal de detalle** al hacer clic en cada elemento: descripción, datos clave y un ejemplo de dónde lo encuentras en la vida real.
- **Buscador en tiempo real**: filtra por nombre, símbolo o número atómico. Los elementos coincidentes se resaltan visualmente con un efecto luminoso.
- **Leyenda interactiva**: cada categoría (No metal, Alcalino, Transición, Gas noble…) funciona como filtro. Al pulsarla, se resaltan los elementos de ese grupo y se atenúan los demás.
- **Tema claro/oscuro automático**: se adapta a las preferencias del sistema operativo (`prefers-color-scheme`).
- **Navegación entre elementos** con botones anterior/siguiente en el modal.
- **Diseño responsive** que se ajusta a distintos tamaños de pantalla.
- **Animaciones** de entrada y efectos de hover con glow por categoría.

## Tecnología

Un único archivo `index.html` sin dependencias externas (salvo fuentes de Google Fonts). HTML + CSS + JavaScript vanilla.

La [primera versión fue creada con Claude Opus 4.6](https://claude.ai/public/artifacts/6e3f9fab-5efb-44d7-9f3d-bf0d3f057105) y posteriormente refinada con Cursor.

## Cómo usarlo

1. Abre `index.html` en cualquier navegador moderno, o visita la [versión publicada](https://javicanton.github.io/tabla-periodica/).
2. Haz clic en cualquier elemento para ver su ficha completa.
3. Usa el buscador para encontrar elementos por nombre, símbolo o número.
4. Pulsa en las categorías de la leyenda para filtrar por tipo de elemento.
